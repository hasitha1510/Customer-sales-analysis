# 🛒 Customer Sales Analysis

> A data analysis project exploring retail customer behavior and purchasing patterns using **Python**, **SQL**, and **Power BI**.

---

## 📌 Project Overview

This project analyzes customer shopping behavior to uncover actionable insights about purchasing patterns, product preferences, and customer segmentation. The goal is to help retail businesses make data-driven decisions to improve sales performance and customer satisfaction.

---

## 🗂️ Repository Structure

```
Customer-sales-analysis/
│
├── customer_shopping_behavior.csv        # Raw dataset
├── Customer_Shopping_Behavior_Analysis.ipynb  # Python EDA & modeling notebook
├── customer_behavior_sql_queries.sql     # SQL queries for insight extraction
├── customer_behavior_dashboard.pbix      # Power BI dashboard file
├── Business Problem Document.pdf         # Project scope and problem statement
└── README.md
```

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| 🐍 Python (Pandas, NumPy, Matplotlib, Seaborn) | Data cleaning, EDA, and modeling |
| 🗄️ SQL (MySQL) | Data extraction and insight queries |
| 📊 Power BI | Interactive dashboard and data visualization |
| 📓 Jupyter Notebook | Analysis workflow and documentation |
| 🐙 GitHub | Version control and project management |

---

## 📊 Key Analysis Areas

- **Customer Segmentation** — Grouping customers by demographics and behavior
- **Purchase Pattern Analysis** — Identifying trends across product categories and time periods
- **Revenue Insights** — Understanding high-value customers and top-performing products
- **Behavioral Trends** — Analyzing frequency, recency, and monetary value (RFM)

## 🚀 Getting Started

### Prerequisites
```bash
pip install pandas numpy matplotlib seaborn jupyter
```

### Run the Notebook
```bash
git clone https://github.com/hasitha1510/Customer-sales-analysis.git
cd Customer-sales-analysis
jupyter notebook Customer_Shopping_Behavior_Analysis.ipynb
```

### SQL Queries
Open `customer_behavior_sql_queries.sql` in your MySQL client and run the queries against your database.

### Power BI Dashboard
Open `customer_behavior_dashboard.pbix` in **Power BI Desktop** to explore the interactive visualizations.

## 📁 Dataset

The dataset (`customer_shopping_behavior.csv`) contains retail transaction records including:
- Customer demographics
- Product categories and purchase amounts
- Transaction dates and frequency
- Shopping preferences and ratings
